# Predicting Residential Real Estate Prices in Ames

## Project Description

The purpose of this study is to predict residential real estate prices in the USA (Ames).  
The data are taken from the Ames Accessor's Office:  
[Ames Housing Dataset](https://www.kaggle.com/datasets/prevek18/ames-housing-dataset)

## Web Application

A web application based on the developed model is available at:  
[Streamlit App](https://real-estate-ames.streamlit.app/)
